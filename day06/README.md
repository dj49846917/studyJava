## 1.断点调试
    <1>.如何加断点:
        在代码区域(有行数的地方)左键双击，出现小点儿

    <2>.运行断点程序:
        代码区域 ==> 右键 ==> debug as ==> java application

    <3>.看哪些区域呢？
		代码区域: 看程序的执行流程
		debug区域: 看程序的执行流程
		variables: 变量的变化
	    console: 看程序的输入和输出

    <4>.如何去掉断点呢？
		1).把加断点的动作再来一遍
 		2).在debug视图中，找到Breakpoints,选中要删除的断点，点击双x即可
