# 字符流和字节流
## 1.标准输出流
        <1>.练习: 读取项目根目录下的FileWriterDemo1.java，并输出到命令行

        <2>.由于标准输出流是一个字节输出流，所以只能输出字节或者字节数组，但是我们读取到的数据是字符串，如果想进行输出还需要转换成字符数组
        我们要想通过标准输出流输出字符串，把标准输出流转换成一种字符输出流即可

        <3>.详细代码请看: 
                day21/code/OutputStreamWriterDemo.java
